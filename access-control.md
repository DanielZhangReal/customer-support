# 权限配置

> ⚠️ 本文件是权限系统的核心，只有 superadmin 可以修改。
> 修改后在对话中说「刷新技能」生效。

## SuperAdmin

只有一个，拥有最高权限。

```
SUPERADMIN_TG_UID: "123456789"
```

## Admin 列表

由 superadmin 动态管理。Admin 可以更新知识库。

```yaml
ADMINS:
  - uid: "987654321"
    name: "客服主管小王"
    added_by: "123456789"
    added_at: "2026-02-26"
  - uid: "111222333"
    name: "运营小李"
    added_by: "123456789"
    added_at: "2026-02-26"
```
