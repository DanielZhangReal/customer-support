# 术语表与名词解释

> 最后更新：2026-02-26
> 维护人：产品 & 客服团队
> 说明：本文档汇总产品中出现的专业术语和内部名词，便于客服准确理解和向用户解释。

---

## 产品相关术语

| 术语 | 解释 | 用户友好说法 |
|------|------|-------------|
| **工作区** | 一个独立的数据隔离空间，不同工作区的数据互不可见 | "您可以理解为不同的项目空间，每个空间的数据是独立的" |
| **成员角色** | 团队成员在工作区内的权限级别，包括拥有者、管理员、成员、只读 | "不同角色有不同的操作权限，管理员可以管理设置，成员可以正常使用" |
| **自动化工作流** | 基于触发条件自动执行的操作规则 | "设定好规则后，系统会自动帮您完成操作，比如新线索自动分配给销售" |
| **仪表盘** | 数据可视化面板，展示关键业务指标 | "一个页面上看到所有重要数据的汇总图表" |
| **API** | 应用程序接口，允许其他系统与我们的产品交互 | "一种技术接口，让您的其他系统可以和我们的产品对接" |
| **Webhook** | 事件触发的回调通知机制 | "当产品内发生某个事件时，自动通知您的其他系统" |
| **SSO** | 单点登录，使用企业统一账号登录 | "用公司统一的账号密码登录，不用单独注册" |
| **SLA** | 服务等级协议，对服务可用性的承诺 | "我们对服务稳定性的保证，如果没达标会有赔偿" |

## 账户与权限术语

| 术语 | 解释 | 用户友好说法 |
|------|------|-------------|
| **拥有者** | 工作区的最高权限角色，可以管理一切设置和成员 | "最高管理员，拥有所有权限" |
| **管理员** | 可以管理成员和大部分设置，但不能删除工作区 | "可以管理团队和设置的管理角色" |
| **成员** | 可以使用所有业务功能，但不能修改设置 | "普通使用者，可以正常使用所有功能" |
| **只读** | 只能查看数据，不能进行任何修改 | "只能看不能改的角色" |
| **二次验证（2FA）** | 登录时除密码外还需验证码的安全措施 | "登录时多一步验证码确认，更安全" |

## 数据相关术语

| 术语 | 解释 | 用户友好说法 |
|------|------|-------------|
| **客户记录** | 系统中一条完整的客户信息 | "一个客户的完整资料卡" |
| **字段** | 客户记录中的单个信息项（如姓名、电话） | "资料卡上的每一项信息" |
| **自定义字段** | 用户自己添加的非默认信息项 | "您可以自己添加需要记录的信息项" |
| **标签** | 用于分类和筛选客户的标记 | "给客户贴的分类标签，方便筛选" |
| **导入/导出** | 批量上传或下载数据 | "批量上传数据到系统 / 从系统下载数据" |
| **数据备份** | 系统自动保存数据副本 | "系统每天自动备份您的数据，不用担心丢失" |

## 支付与账单术语

| 术语 | 解释 | 用户友好说法 |
|------|------|-------------|
| **订阅** | 按月或按年付费使用服务 | "按月或按年付费来使用产品" |
| **账单周期** | 两次扣款之间的时间间隔 | "每次扣费之间的间隔，月付就是一个月" |
| **宽限期** | 订阅到期后仍可使用的缓冲时间（7 天） | "到期后有 7 天缓冲期，续费后可以继续正常使用" |
| **按比例折算** | 升级/降级时按已使用天数计算费用差额 | "升级时只需补剩余天数的差价" |
| **优惠码/兑换码** | 可抵扣费用或激活特定权益的代码 | "输入后可以享受折扣或激活特定功能" |

## 技术 / 安全术语

| 术语 | 解释 | 用户友好说法 |
|------|------|-------------|
| **TLS** | 传输层安全协议，保护数据在传输过程中不被窃取 | "数据在传输过程中是加密的，别人看不到" |
| **AES-256** | 一种高强度加密算法 | "银行级别的数据加密保护" |
| **ISO 27001** | 国际信息安全管理体系认证 | "通过了国际权威的信息安全认证" |
| **SOC 2** | 服务组织安全性审计标准 | "经过专业机构审计的安全标准" |
| **渗透测试** | 模拟黑客攻击来检测系统安全漏洞 | "定期请安全专家来测试我们系统的安全性" |
| **CDN** | 内容分发网络，加速全球访问速度 | "让各地用户都能快速访问，不卡顿" |
| **状态页** | 展示系统运行状态的公开页面 | "可以查看系统是否正常运行的页面：status.example.com" |

## 客服内部术语（不对用户使用）

| 术语 | 含义 |
|------|------|
| **工单** | 用户问题的跟踪记录 |
| **SOP** | 标准操作流程 |
| **升级（Escalation）** | 将问题转交给更高级别的支持人员 |
| **首响时间** | 用户提问到客服首次回复的时间 |
| **解决率** | 问题被成功解决的比例 |
| **CSAT** | 客户满意度评分 |
| **NPS** | 净推荐值，衡量用户忠诚度 |
| **Churn** | 客户流失 |
| **MRR** | 月经常性收入 |
